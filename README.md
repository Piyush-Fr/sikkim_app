# Sikkim Tourism App

A Flutter-based tourism application for exploring **Sikkim**, designed to help users discover destinations, navigate through maps, and interact with an in-app chatbot for travel-related assistance.

## Overview

The **Sikkim Tourism App** is a cross-platform Flutter project that provides travelers and explorers with a digital guide to Sikkim. The app is built to make tourism information more accessible and interactive through a clean interface, integrated maps, and a chatbot that can assist users with travel queries and guidance.

Whether someone is planning a trip or already visiting Sikkim, this app aims to provide useful tourism support directly on mobile devices.

## Features

- **Tourism Information**
  - Explore places and travel-related content about Sikkim
  - Access useful destination details in one place

- **Interactive Map**
  - View locations through an integrated map
  - Improve navigation and place discovery

- **In-App Chatbot**
  - Ask tourism-related questions
  - Get guidance and support while exploring the app

- **Cross-Platform Support**
  - Built with Flutter for multiple platforms including Android, iOS, Web, Windows, macOS, and Linux

- **Scalable Architecture**
  - Structured code organization using folders like `config`, `screens`, and `services`

## Tech Stack

- **Flutter**
- **Dart**
- **Firebase**
- **Maps Integration**
- **Chatbot Functionality**
